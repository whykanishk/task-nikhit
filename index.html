Recipie Web Application

A simple web app where you can search for a recipie via its name (title) or ingrediants. By defalt this web App show all recipies and its ingrediants.

  • Peanut-butter & Jelly Sandwich: Peanut butter, Jelly, Bread
  • Buckeyes: Chocolate, Peanut butter, Sugar
  • Hawaiian Pizza: Cheese, Pizza Dough, Tomato Sauce, Pineapple, Bacon
  • Hot Chocolate: Milk, Chocolate, Sugar
  • Grilled Cheese: Sandwich bread, Cheese
